## What is FRIDAY?
AI Assistant inspired by Iron Man JARVIS. Built on Android phone using Termux and Python. No laptop needed!

## Features
- AI Chat with Groq LLaMA 3.3 70B
- Live Weather with 5-day forecast
- Live News - World Tech Sports Business
- NASA Space fact of the day
- Jokes and Quotes in Roman Hindi
- IP Lookup and Network info
- Night Guard Security System
- Voice TTS Iron Man UI
- 8387 lines single Python file

## Install
pkg install python git mpv termux-api -y
pip install groq requests gtts --break-system-packages
export GROQ_API_KEY=your_key_here
python miraz.py
